The journey so far...

My series of cheatsheets have led me through the design process of laying out and positioning elements on the page, and how best to select the elements to apply styling. Now that I have my building blocks in place it is time to start with the actual styles that can be applied....starting with the background!

A background applied to an element sits under the content, padding and border by default (margin doesn't count as part of an element's area) and is on the bottom of the z-index stack.

The properties for background are:

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-attachment
  • background-size
  • background
  • background-clip

Background Color

Did you know that the default background colour is transparant and not white!

It is recommended to always use a background color, even as just a fallback if the browser doesnâ€™t support gradients or link to image is broken.

.top {
background-color: #f3f2f1;
}

Background Image

After setting a fallback with background-color an image can be set using background-image to appear as the background

All background images will repeat by default so it looks like a tile affect

Screen readers will not register any images applied this way using CSS

.top {
background-color: #f3f2f1; /*fallback*/
background-image: url(https://imagehosting/image.png);
}

Colour gradients can also be set using background-image.

.top {
background-color: #f3f2f1; /*fallback*/
background-image: linear-gradient(to bottom, orange, blue, white); }

Multiple images can be set using background-image.

.top {
background-color: #f3f2f1; /*fallback*/
background-image: linear-gradient(to bottom, orange, blue, white);/*fallback*/
background-image: url(https://imagehosting/image.png); }
Color Gradients

Gradients let you display smooth transitions between two or more specified colours.

There are two types of color gradient that can be used as a background image:

  1. Linear
  2. Radial

Linear Gradient

It works pretty much as it sounds, the linear gradient can be from up to down, side to side, or diagonal.

To create a linear gradient you need:

  1. A direction or angle (default alignment is from top to bottom)
  2. Two or more colours (transparant counts as a colour!)
/*direction set using default of top to bottom*/
div {
background-image: linear-gradient(color1, color2);
}
/*direction set from left to right*/
div {
background-image: linear-gradient(to right, color1, color2, color3);
}
/*direction set diagonally*/
div {
background-image: linear-gradient(to top left, color1, color2, color3, color4);
}

For more control of the direction of the gradient an angle can be set instead of the directions

/*direction set as angle 0deg which is same as bottom to top*/
div {
background-image: linear-gradient(0deg, color1, color2, color3, color4, color5);
}
/*direction set as angle 90deg which is same as left to right*/
div {
background-image: linear-gradient(90deg, color1, color2, color3, color4, color5, color6);
}
/*direction set as angle 180deg which is same as top to bottom*/
div {
background-image: linear-gradient(180deg, color1, color2, color3, color4, color5, color6, color7);
}

This allows more exact diagonal lines to be used...

/*direction set as angle 125deg */
div {
background-image: linear-gradient(125deg, color1, color2, color3);
}
/*direction set as angle 270deg */
div {
background-image: linear-gradient(270deg, color1, color2, color3);
}

The gradient can also be set to repeat itself...

/*2 color gradient set to repeat*/
div {
background-image: repeating-linear-gradient(0deg, color1, color2);
}

Radial Gradient

The radial gradient radiates from a central point.

To create a radial gradient you need:

  1. A shape
  2. Two or more colours (transparant counts as a colour!)

There are two shapes:

  1. Ellipse (default)
  2. Circle
/*3 color elliptical radial gradient*/
div {
background-image: radial-gradient(red, yellow, green);
}
/*2 color gradient set to repeat*/
div {
background-image: repeating-radial-gradient(color1, color2);
}

The size parameter defines the size of the gradient.

The values are:

  • closest-side
  • farthest-side
  • closest-corner
  • farthest-corner
/*3 color radial gradient using closest-side size*/
div {
background-image: radial-gradient(closest-side at 70% 25%, color1, color2, color3);
}
/*3 color radial gradient using farthest-side size*/
div {
background-image: radial-gradient(farthest-side at 70% 25%, color1, color2, color3);
}
/*3 color radial gradient using closest-corner size*/
div {
background-image: radial-gradient(closest-corner at 70% 25%, color1, color2, color3);
}
/*3 color radial gradient using farthest-corner size*/
div {
background-image: radial-gradient(farthest-corner at 70% 25%, color1, color2, color3);
}

Colour stops

For both linear and radial gradients you can also specify how much of the gradient is in each colour instead of all using an equal fraction....

/*3 color circular linear gradient*/
div {
background-image: linear-gradient(to right, color1 25%, color2 50%, color3);
}
/*3 color circular radial gradient*/
div {
background-image: repeating-linear-gradient(color1 20%, color2 15%, color3);
}

Background Repeat

The background-repeat property allows you to specify how the background image is repeated. The default is that an image is both horizontally and vertically repeated.

Values that can be set are:

  • no-repeat
  • repeat-x
  • repeat-y
  • repeat (default)
.top {
background-color: #f3f2f1; /*fallback*/
background-image: url(https://imagehosting/image.png);
background-repeat: no-repeat;
}

Background Position

The background-position property allows you to position the background image where you want in the background by specifying an x(horizontal) and y (vertical) value (top left being (0,0)).

Values can be set using:

  • pixels
  • rems
  • percentages
  • keywords (left, center, right, top, center, bottom)
div{
background-color: #f3f2f1; /*fallback*/
background-image: linear-gradient(to bottom, orange, blue, white); /*fallback*/
background-image: url(https://imagehosting/image.png);
background-repeat: no-repeat;
background-position: 30px 50px;
}
/*background top and center*/
div {
background-image: url(https://imagehosting/image.png);
background-repeat: no-repeat;
background-position: top center;
}

Background Attachment

The background-attachment property allows you to specify how to scroll when the content scrolls if the content will scroll.

Values that can be set are:

  • scroll
  • fixed
  • local

Scroll

If the element is scrolled the background does not move and scrolls as the page scrolls

Fixed

Always remains fixed to the viewport

Local

Fixes the background to the element it is set on and when element is scrolled and background scrolls with it

div{
background-color: #f3f2f1; /*fallback*/
background-image: linear-gradient(to bottom, orange, blue, white); /*fallback*/
background-image: url(https://imagehosting/image.png);
background-repeat: no-repeat;
background-position: 30px 50px;
background-attachment: local;
}

Background Size

The background-size property does pretty much exactly what you think it does (but check browser support)

Values can be set as

  • pixels
  • rems
  • percentages
div{
background-color: #f3f2f1; /*fallback*/
background-image: linear-gradient(to bottom, orange, blue, white); /*fallback*/
background-image: url(https://imagehosting/image.png);
background-repeat: no-repeat;
background-position: 30px 50px;
background-attachment: local;
background-size: 160px 160px;
}

Background

The background property is the shorthand of all above property values (any properties not specified are assigned default values).

So this....

header {
background-color: yellow;
background-image: linear-gradient(to bottom, yellow, #dddd00 50%, orange);
background-repeat: no-repeat;
background-position: 99% center;
}

can be written as...

header { background: yellow linear-gradient(to bottom, orange, yellow) no-repeat left center scroll; }

This is where adding multiple background can get fun..

header {
background-color: yellow;
background: url(image.png) no-repeat 99% center, url(background-tile.png),
linear-gradient(to bottom, yellow, #dddd00 50%, orange);
}
